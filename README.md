# NextLevelCoder

A full-stack learning and coding platform built using:

- React
- TypeScript
- Spring Boot
- MySQL
- Tailwind CSS

## Features

- User Authentication
- Dashboard
- Problem Solving
- Profile Management
- Settings
- Topic Learning

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Spring Boot
- Java

### Database
- MySQL

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
mvn spring-boot:run
```
